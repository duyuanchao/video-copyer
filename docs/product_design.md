# Video Copyer 产品设计文档

## 1. 项目概述 (Project Overview)

**Video Copyer** 旨在打造一个基于 Web 的 AI 视频内容创作平台，通过复制“爆款”逻辑，帮助普通用户、自媒体创作者及中小型商家快速生成高质量的短视频内容。

核心理念是“理解即复刻”，通过 AI 深度解析热门视频的结构、节奏和文案与视觉风格，实现从“看懂”到“再创作”的闭环。

## 2. 核心功能 (Core Features)

### 2.1 爆款视频复刻 (Viral Video Replica)
这是平台的核心差异化功能。
*   **输入**: 支持粘贴抖音/TikTok 视频链接，或直接上传本地视频文件。
*   **AI 解析**:
    *   **结构拆解**: 分析视频的起承转合、黄金前三秒、情绪钩子。
    *   **元素提取**: 提取文案脚本、分镜描述、背景音乐风格、运镜节奏。
    *   **DNA 提取**: 总结该视频成为“爆款”的核心逻辑（如：反转剧情、快节奏科普、治愈系画面）。
*   **再创作**:
    *   **主题重构**: 用户输入新的主题（如将“历史科普”复刻为“户外运动”）。
    *   **智能生成**: 生成全新的脚本、分镜画面、配音。
    *   **二次编辑**: 提供简单的非线性编辑界面，允许用户替换关键帧、修改文案、调整配音。

### 2.2 照片会说话 (Talking Photos)
*   **输入**: 上传单张或多张人物/宠物/静物照片。
*   **剧本编排**: 输入对话文本或上传音频。
*   **多角色对话**: 支持为不同图片指定角色和台词，AI 自动生成多角色对话视频。
*   **表现力**: 自动匹配口型、眼神交互和肢体微动作，减少“僵硬感”。

### 2.3 智能生视频 (Intelligent Video Generation)
*   **输入**: 文本 Prompt 或 参考图片。
*   **工作流**:
    1.  **理解分析**: AI 分析用户意图和受众群体。
    2.  **创意策略**: 生成 2-3 个创意方向供用户选择。
    3.  **剪辑成片**: 生成最终视频，融合 AI 生成素材与用户上传素材。

## 3. 用户体验与设计 (UX/UI Design)

*   **视觉风格**: 采用“极简”且“科技感”的设计。
    *   **配色**: 深色模式为主（Dark Mode），搭配霓虹色（青色/紫色）作为强调色，体现 AI 的先进感。
    *   **质感**: 玻璃拟态（Glassmorphism）卡片，微动效（Hover effects）。
*   **交互原则**:
    *   **向导式流程**: 复杂的 AI 任务通过 Chat 或分步 Wizard 引导。
    *   **即时反馈**: 在处理耗时任务时，展示清晰的进度条和有趣的 Loading 动画（如“AI 正在拆解爆款因子...”）。

## 4. 技术架构方案 (Technical Architecture)

### 4.1 前端 (Frontend)
*   **核心框架**: **Next.js 14+** (App Router) - 提供最佳的性能和 SEO 支持。
*   **语言**: TypeScript.
*   **样式**: **Tailwind CSS** - 快速构建现代 UI。
*   **组件库**: Shadcn/ui (基于 Radix UI) + Framer Motion (动画)。
*   **状态管理**: Zustand 或 TanStack Query (React Query).

### 4.2 后端与 AI (Backend & AI)
*   *初期由于是 Web 版本，我们将主要依赖 API 集成。*
*   **API 路由**: 使用 Next.js API Routes 处理轻量级逻辑。
*   **多媒体处理**: Ffmpeg.wasm (前端处理) 或 云端 Serverless Function (处理视频流)。
*   **AI 模型集成 (模拟/接入)**:
    *   *视频理解*: 接入多模态大模型 API (如 GPT-4o, Gemini 1.5 Pro) 用于视频内容分析。
    *   *图像/视频生成*: 接入 Stable Diffusion, Runway, 或 Midjourney API。
    *   *语音合成*: 接入 OpenAI TTS, ElevenLabs。
    *   *数字人*: 接入相关数字人 API (如 D-ID, HeyGen) 或使用开源模型 (SadTalker)。

## 5. 开发路线图 (Development Roadmap)

### Phase 1: 基础框架与 UI (MVP)
*   搭建 Next.js 项目脚手架。
*   实现 Landing Page (着陆页)，展示三大功能入口。
*   构建“工作台”布局 (Sidebar + Main Content)。

### Phase 2: 爆款复刻核心流程
*   实现“视频上传/解析”的前端模拟流程。
*   设计“视频拆解报告”界面 (展示文案、结构)。
*   实现“二次编辑”简易编辑器 (基于 Timeline 的 UI)。

### Phase 3: 完善其他功能
*   开发“照片会说话”交互界面。
*   开发“智能生视频”的 Prompt 输入与结果展示页。

### Phase 4: 优化与特效
*   增加全站微交互与动画。
*   适配移动端响应式布局。
